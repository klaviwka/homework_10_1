# Проект: Маскировка данных и обработка транзакций

## Цель проекта

Данный проект включает функции для маскировки номеров кредитных карт и счетов, преобразования формата даты, а также фильтрации и сортировки транзакций. Эти функции могут быть полезны для обеспечения безопасности данных пользователей и упрощения работы с финансовыми записями.

## Использование

### 1. Маскировка номера кредитной карты

Функция get_mask_card_number маскирует номер кредитной карты, оставляя видимыми только первые 6 и последние 4 цифры.


### 2. Маскировка номера счета

Функция get_mask_account маскирует номер счета, оставляя видимыми только последние 4 цифры.


### 4. Преобразование формата даты

Функция get_date преобразует строку с датой из формата 'YYYY-MM-DDTHH:MM:SS.ssssss' в формат 'DD.MM.YYYY'.


### 5. Фильтрация транзакций

Функция filter_by_state фильтрует список транзакций по значению ключа 'state'.


### 6. Сортировка транзакций по дате

Функция sort_by_date сортирует список транзакций по дате.
